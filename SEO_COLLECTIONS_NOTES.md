# SEO Scale Expansion

Estruturas de páginas para crescimento orgânico:

## Páginas recomendadas

- jogos fáceis de platinar
- platinas rápidas (até 10h / 20h / 30h)
- jogos difíceis de platinar
- melhores platinas para iniciantes
- jogos com troféus perdíveis
- jogos sem troféus online
- ranking por gênero (RPG, Soulslike, Indie, Horror)
- ranking por plataforma
- ranking por dificuldade

## Regras importantes

- cada página precisa de texto introdutório real
- evitar páginas vazias sem contexto
- canonical correto
- schema + breadcrumbs
- links internos fortes para páginas de jogo

## Estrutura editorial

1. resumo da coleção
2. critérios da lista
3. cards dos jogos
4. perguntas frequentes
5. links relacionados
